## 배운 점
- `application.yml`은 Spring Boot 실행 환경 및 설정값을 관리하는 핵심 설정 파일이다.
- 환경변수는 보안을 위해 설정값을 코드와 분리하는 데 유용하다. 
- JPA는 객체와 데이터베이스를 연결하는 기술이고, Auditing은 생성 시각과 수정 시각 같은 공통 이력값을 자동으로 관리하는 데 사용된다.
